# Superpower Kanban — Claude Code Plugin Marketplace

Custom Claude Code plugins by OrionLabz.

## Plugins

### task-viewer

Real-time Kanban dashboard for Claude Code sessions with Superpowers plan tracking.

**Install:**

```bash
claude plugins add-marketplace orionlabz/superpower-kanban
claude plugins add task-viewer --marketplace orionlabz/superpower-kanban
```

**Update:**

```bash
claude plugins update task-viewer
```
